Пояснение

В папке functions сделать функцию с названием "getUsers", которая будет делать запрос на url="https://jsonplaceholder.typicode.com/users" и получать список пользователей, которые нужно хранить в state. State - локальное хранилище данных для минимизации запросов, что облегчает работу сайта и ускоряет его загрузку. Чтобы поместить данные в state нужно использовать соответствующий плагин в папке plugins.
